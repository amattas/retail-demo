# Notebooks

Fabric notebooks for ingestion, Silver/Gold processing, ML, and manual Lakehouse administration.

Key notebook groups:
- `01-05` core Bronze/Silver/Gold setup, streaming, and maintenance
- `06-14` ML and advanced analytics notebooks
- `30-create-ontology.ipynb` manual Fabric ontology creation from core Silver retail tables
- `99-reset-lakehouse.ipynb` optional reset utility for rebuilding the Lakehouse
