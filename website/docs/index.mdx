A Microsoft Fabric Real-Time Intelligence demo for the retail industry. A
synthetic data generator produces historical and streaming retail data, and
Fabric assets (Eventhouse/KQL Database, Lakehouse medallion layers,
dashboards, and a Power BI semantic model) turn it into real-time analytics,
alerting, and decision support.

## What's inside

- **Streaming notebook** writes events directly to KQL tables with the Fabric Spark connector for Kusto
- **KQL Database** (`retail_eventhouse`) with event tables, functions, and materialized views for hot KPIs
- **Lakehouse** Bronze → Silver → Gold medallion layers for persistence and history
- **Real-Time Dashboards** and querysets over the materialized views
- **Power BI semantic model** with curated DAX measures and hierarchies
- **Synthetic data generation** through Fabric setup notebooks and optional live streaming to Eventhouse
